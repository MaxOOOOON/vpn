    tun;

---

## Выполнение

Подключение к серверу на клиенте

    openvpn /etc/openvpn/client/tun.ovpn &

либо 

    openvpn /etc/openvpn/client/tap.ovpn &

Проверка с локальной машины

    openvpn buffer/tap.ovpn
